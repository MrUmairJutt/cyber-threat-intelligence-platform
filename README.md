TITLE

AI-Powered Cyber Threat Intelligence Platform

DESCRIPTION

This project is designed to detect and analyze cyber threats using Machine Learning and behavior analysis. It processes data from network logs and sample datasets to identify suspicious activities and generate alerts.

FEATURES

• Real-time threat detection
• Risk scoring system
• Alert generation
• Data visualization dashboard

TECHNOLOGIES USED

• Python
• Machine Learning
• Basic Data Analysis

STATUS

This is an academic Final Year Project (FYP) developed for learning and research purposes.
